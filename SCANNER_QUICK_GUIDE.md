## 🐛 Troubleshooting

### Camera not working?
**Check:**
- Camera permissions granted
- Using HTTPS (required for camera)
- Camera not used by another app
- Browser supports camera API

**Try:**
- Reload page and allow camera
- Check browser settings
- Try different browser
- Restart browser

### Scanner not detecting?
**Check:**
- QR code is clear and visible
- Good lighting conditions
- Hold steady for 1-2 seconds
- QR not damaged or blurry

**Try:**
- Move camera closer/farther
- Improve lighting
- Clean camera lens
- Use manual input as backup

### Handheld scanner not working?
**Check:**
- Scanner properly connected
- Scanner LED shows ready
- Scanner configured for keyboard mode
- Focus in scanner input area
