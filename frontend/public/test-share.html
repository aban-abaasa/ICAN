🧪 Test Share Target

This page simulates sharing content to IcanEra. Use it to test the share functionality without needing another app.

ℹ️ How to Use

  1. Fill in the form fields (or use the defaults)
  2. Optionally select an image file
  3. Click "Test Share to IcanEra"
  4. The app should open with the Status Uploader pre-filled

Note: The service worker must be registered for this to work properly.
